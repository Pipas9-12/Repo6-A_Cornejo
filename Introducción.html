¿Qué es ARINC 429?

ARINC 429 es el sistema que usan las aeronaves para que sus distintas partes electrónicas puedan comunicarse entre sí. Por ejemplo, así se envían datos como la altitud, el rumbo o la velocidad entre los instrumentos del avión.

¿Para qué se usa?

Sirve para enviar información de forma segura y ordenada, evitando interferencias, de un equipo a otro dentro del avión. Así, todo funciona bien y los pilotos reciben los datos correctos.

¿Cómo funciona?

ARINC 429 envía mensajes que siempre tienen 32 “bits” . Esos bits dicen qué tipo de dato se está mandando (como rumbo, altitud, etc) y cuál es su valor.

Cada equipo puede enviar datos a muchos otros receptores(hasta 20), y los datos pueden enviarse muy seguido o solo cuando haga falta. Además, este sistema tiene un control que revisa si los datos llegan bien.

¿Qué tan rápido trabaja?

Algunos datos importantes se mandan muy seguido, por ejemplo cada 50 milisegundos. Otros menos urgentes, cada medio segundo(500 milisegundos). La velocidad de transmisión puede ser rápida (100 kbps) o más lenta (12-14.5 kbps).

¿Cómo se escriben los datos?

Los datos se mandan en diferentes formas según el tipo:

  • BNR: Para valores como altitud o temperatura.
  • BCD: Para números con decimales, como los grados del rumbo.
  • Discretos: Para cosas que están prendidas o apagadas, como interruptores.

¿Por qué es tan usado?

Porque hace que los equipos de distintas funciones y marcas puedan trabajar juntos dentro del avión. Gracias a esto, los sistemas son más seguros y compatibles entre sí, algo muy importante para volar.
